# 数组排序之选择排序和冒泡排序

## 1，选择排序

```
for(int i=0;i<a.length;i++){
    for(int j=i+1;j<a.length;j++){
        if(a[i]>a[j]){
            int temp = a[i];
            a[i] = a[j];
            a[j] = temp;
        }
    }
}
```

## 2，冒泡排序

```
for(int i=0;i<a.length;i++){
    for(int j=0;j<a.length-i-1;j++){
        if(a[j]>a[j+1]){
            int temp = a[j];
            a[j] = a[j+1];
            a[j+1] = temp;
        }
    }
}
```

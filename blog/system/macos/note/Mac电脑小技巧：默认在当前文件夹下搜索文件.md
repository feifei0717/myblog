# Mac电脑小技巧：默认在当前文件夹下搜索文件

天狐 2017-06-15 21:22

在macOS 系统中 Finder 中的搜索框进行搜索时，默认是在”这台 Mac”下搜索，也就是说会在整个系统中搜索，很费时间的。而我们往往只想在当前目录下搜索，很麻烦。 如下配置后，就可以让 Mac 默认在当前文件夹下搜索了
